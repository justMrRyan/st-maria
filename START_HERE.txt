❓ FAQ:

  Q: Where do I get my Supabase credentials?
  A: supabase.com → Create project → Settings > API

  Q: How do I set up Coflow OAuth?
  A: Register app with your Coflow provider, get Client ID & Secret

  Q: Can I change the design?
  A: Yes! Edit app/globals.css and component files

  Q: Can I use a different database?
  A: Requires code changes, but architecture is similar

  Q: How do I deploy?
  A: Push to GitHub → Connect to Vercel → Add env vars → Deploy

🆘 IF SOMETHING BREAKS:

  Problem: "supabaseUrl is required"
  Solution: Check .env.local exists with credentials

  Problem: "Not authenticated"
  Solution: Check Coflow credentials match your app

  Problem: Images won't upload
  Solution: Make sure storage bucket is PUBLIC, not private

  Problem: Can't access dashboard
  Solution: Check OWNER_EMAIL matches your login email

  → Full troubleshooting guide in SETUP_GUIDE.md
